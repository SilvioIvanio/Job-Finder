# WorkNet - Job Finder Application

A desktop application built with Java Swing that helps connect job seekers with employers. This project is currently in its initial development phase, starting with a robust login and registration system.

## Project Overview

WorkNet is designed to be a comprehensive job finder system that will allow:
- Job seekers to find and apply for positions
- Employers to post job opportunities and review applications
- Simple and intuitive user interface for both parties

## Current Implementation

The current codebase includes:
- Login/Sign up system with a tabbed interface
- User type selection (Job Seeker/Employer)
- Basic form validation
- Placeholder for database integration
- Separate dashboard views for different user types (planned)

## Technical Stack

- **Language:** Java (JDK 11 or later)
- **GUI Framework:** Java Swing
- **Database:** MySQL (planned)
- **Build Tool:** Maven/Gradle (to be implemented)
- **IDE Support:** IntelliJ IDEA, Eclipse, VS Code

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 11 or later
- Your preferred Java IDE
- Git for version control

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/username/WorkNet.git
   ```
2. Open the project in your IDE
3. Run `LoginPage.java` to start the application
